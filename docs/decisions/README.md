# Task Decision Records

Use this directory for task-level planning and implementation notes.

Each task should get one folder:

```text
docs/decisions/YYYY-MM-DD-task-slug/README.md
```

The task record should include:

- Goal
- Decisions
- Plan
- Implementation notes
- Validation
- Follow-ups

These records replace the old requirement that every development session must start from a GitHub Issue. GitHub Issues can still be referenced when useful, but they are optional.
