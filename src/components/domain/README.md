# Domain Components

This directory owns SnipForge-specific components. Domain components may compose `src/components/ui` primitives and may know about commands, libraries, settings, Electron APIs, or app shell behavior.

Groups:

- `commands/`: command workflow UI such as variable prompts and duplicate handling.
- `libraries/`: library-specific surfaces and list items.
- `settings/`: settings-specific sections and rows.
- `shell/`: app shell UI such as help, description, and update surfaces.
